# Hiển thị một ngày trong tuần

Viết hàm `getWeekDay(date)` để hiển thị ngày trong tuần ở định dạng ngắn: 'MO', 'TU', 'WE', 'TH', 'FR', 'SA', 'SU'.

Ví dụ:

```js no-beautify
let date = new Date(2012, 0, 3);  // 3 Thg 1 2012
alert( getWeekDay(date) );        // nên xuất ra "TU"
```
