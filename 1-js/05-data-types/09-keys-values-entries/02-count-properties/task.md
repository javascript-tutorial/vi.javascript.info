# Đếm thuộc tính

Viết hàm `count(obj)` trả về số thuộc tính trong đối tượng:

```js
let user = {
  name: 'John',
  age: 30
};

alert( count(user) ); // 2
```

Cố gắng làm cho mã càng ngắn càng tốt.

Tái bút: Bỏ qua các thuộc tính tượng trưng, chỉ tính những thuộc tính "thông thường".
